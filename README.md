## 📌 Project Overview

This project was completed during my **Data Technician Bootcamp**, focusing on building a complete Business Intelligence solution using Power BI.

The objective was to take raw retail data and transform it into an interactive dashboard that supports business decision-making through clear and meaningful insights.

The final report explores sales performance, customer behaviour, and product trends across multiple dimensions.

---

## 🎯 Key Objectives

- Analyse retail sales performance across regions  
- Identify customer and product trends  
- Track revenue changes over time  
- Compare categories and segments  
- Enable interactive filtering for deeper analysis  

---

## 🧹 Data Cleaning & Transformation (Power Query)

Before analysis, the dataset was prepared using Power Query to ensure it was accurate and consistent.

### Key steps included:
- Removing missing values and correcting data errors  
- Standardising column formats and data types  
- Splitting and merging fields for better structure  
- Creating new calculated columns  
- Shaping data into a model-ready format  

This ensured a clean and reliable dataset for reporting.

---

## 🧠 DAX Measures & Analytics

To enhance the depth of analysis, I created several DAX measures to support dynamic reporting.

### Key calculations included:
- 📅 Year-to-Date (YTD) and Month-to-Date (MTD) metrics  
- 💰 Total Sales, Profit, and Quantity measures  
- 🌍 Region and category-level aggregations  
- 📊 Percentage change and performance comparisons  

These measures allowed the dashboard to respond dynamically to filters and user interaction.
